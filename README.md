# TidyFlow

Desktop file organizer built with Next.js 15 + Tauri v2.

## Local Run

1. Install dependencies:
```bash
bun install
```
2. Web app only:
```bash
bun run dev
```
3. Desktop app (requires Rust/Cargo installed):
```bash
bun run tauri:dev
```

## Build Windows Installer Without Local Rust

This repo includes a GitHub Actions workflow that builds installer artifacts (`.exe` and `.msi`) in the cloud.

### Option A: Manual build from GitHub UI

1. Push this project to GitHub.
2. Open `Actions` tab.
3. Run workflow: `Build Windows Installer`.
4. Download artifact: `tidyflow-windows-installers`.

### Option B: Build + auto-release on version tags

1. Create and push a tag:
```bash
git tag v0.1.0
git push origin v0.1.0
```
2. Workflow builds installers and attaches them to the GitHub Release for that tag.

## Installer Output

- NSIS installer: `TidyFlow_*_x64-setup.exe`
- MSI installer: `TidyFlow_*_x64_en-US.msi`

Generated by:
- `.github/workflows/windows-installer.yml`
